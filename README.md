## 📦 Featured Projects

### 🧩 [Enterprise Component Platform for Java](https://github.com/pbxjtumse/cola-web-standard)

An enterprise-grade Java infrastructure component platform built with Domain-Driven Design principles and the COLA layered architecture.

The project aims to standardize common infrastructure capabilities, reduce duplicated development across business systems, and gradually establish a reusable, extensible, observable, and governable engineering ecosystem.

[![Repository](https://img.shields.io/badge/GitHub-cola--web--standard-181717?logo=github&logoColor=white)](https://github.com/pbxjtumse/cola-web-standard)
[![Components](https://img.shields.io/badge/Source-components-0969DA?logo=github&logoColor=white)](https://github.com/pbxjtumse/cola-web-standard/tree/master/component)

#### Core Capabilities

- Multi-level caching, cache penetration protection, and hot-key management
- Unified thread-pool management and asynchronous task orchestration
- Rejection handling, timeout control, cancellation, fallback, and retry support
- Distributed locks, lease renewal, ownership validation, and fencing-token design
- Message production, consumption, acknowledgment, retry, dead-letter, and ordering control
- Observability, service governance, and configuration management
- Distributed transactions, idempotency control, and consistency guarantees
- Spring Boot Starter auto-configuration and out-of-the-box integration
- Production-oriented unit testing, concurrency testing, and performance validation

## 📦 代表项目

### 🧩 [企业级基础组件平台（Java）](https://github.com/pbxjtumse/cola-web-standard)

基于领域驱动设计和 COLA 分层架构构建的企业级 Java 基础组件工程。

项目目标是统一封装业务系统中常见的基础技术能力，减少重复建设，并逐步沉淀一套可复用、可扩展、可观测、可治理的企业级工程体系。

[![Repository](https://img.shields.io/badge/GitHub-cola--web--standard-181717?logo=github&logoColor=white)](https://github.com/pbxjtumse/cola-web-standard)
[![Components](https://img.shields.io/badge/Source-components-0969DA?logo=github&logoColor=white)](https://github.com/pbxjtumse/cola-web-standard/tree/master/component)

#### 核心能力

- 多级缓存、缓存穿透保护与热点数据治理
- 统一线程池管理与异步任务编排
- 拒绝处理、超时控制、任务取消、回退与重试
- 分布式锁、租约续期、持有者校验与 Fencing Token 设计
- 消息发送、消费、确认、重试、死信与顺序控制
- 可观测性、服务治理与配置能力集成
- 分布式事务、幂等控制与数据一致性保障
- Spring Boot Starter 自动装配与开箱即用
- 面向生产环境的单元测试、并发测试与性能验证
