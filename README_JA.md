## 📦 主要プロジェクト

### 🧩 [Java エンタープライズ基盤コンポーネントプラットフォーム](https://github.com/pbxjtumse/cola-web-standard)

ドメイン駆動設計と COLA レイヤードアーキテクチャを採用して構築した、エンタープライズ向け Java 基盤コンポーネントプロジェクトです。

業務システムで共通して利用される基盤機能を標準化し、重複開発を削減するとともに、再利用性、拡張性、可観測性、ガバナンス性を備えたエンジニアリング体系を構築することを目的としています。

[![Repository](https://img.shields.io/badge/GitHub-cola--web--standard-181717?logo=github)](https://github.com/pbxjtumse/cola-web-standard)
[![Components](https://img.shields.io/badge/Source-components-blue?logo=github)](https://github.com/pbxjtumse/cola-web-standard/tree/master/component)

#### 主な機能

* 多段キャッシュ、キャッシュペネトレーション対策、ホットキー管理
* スレッドプールの統合管理と非同期タスクのオーケストレーション
* 拒否処理、タイムアウト制御、キャンセル、フォールバック、リトライ
* メッセージ送受信、ACK、リトライ、デッドレター、順序制御
* オブザーバビリティ、サービスガバナンス、設定管理
* 分散トランザクション、冪等性制御、データ整合性の保証
* Spring Boot Starter による自動設定と容易な導入
* 本番環境を想定した単体テスト、並行テスト、性能検証
