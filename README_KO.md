## 📦 주요 프로젝트

### 🧩 [Java 엔터프라이즈 기반 컴포넌트 플랫폼](https://github.com/pbxjtumse/cola-web-standard)

도메인 주도 설계와 COLA 계층형 아키텍처를 기반으로 구축한 엔터프라이즈급 Java 기반 컴포넌트 프로젝트입니다.

비즈니스 시스템에서 공통적으로 사용하는 인프라 기능을 표준화하고, 중복 개발을 줄이며, 재사용성, 확장성, 관측 가능성, 거버넌스 역량을 갖춘 엔지니어링 체계를 구축하는 것을 목표로 합니다.

[![Repository](https://img.shields.io/badge/GitHub-cola--web--standard-181717?logo=github)](https://github.com/pbxjtumse/cola-web-standard)
[![Components](https://img.shields.io/badge/Source-components-blue?logo=github)](https://github.com/pbxjtumse/cola-web-standard/tree/master/component)

#### 핵심 기능

* 다단계 캐시, 캐시 관통 방지, 핫키 관리
* 통합 스레드 풀 관리와 비동기 작업 오케스트레이션
* 거부 처리, 타임아웃 제어, 취소, 폴백, 재시도
* 메시지 송수신, ACK, 재시도, 데드 레터, 순서 제어
* 옵저버빌리티, 서비스 거버넌스, 설정 관리
* 분산 트랜잭션, 멱등성 제어, 데이터 일관성 보장
* Spring Boot Starter 자동 설정과 간편한 통합
* 운영 환경을 고려한 단위 테스트, 동시성 테스트, 성능 검증
